# Seed Maker 화면 목록

## 정보 구조 (IA)

```
Seed Maker
├── / (메인 대시보드)
├── /expenses (지출 목록)
├── /expenses/new (지출 추가)
├── /budget (예산 설정)
├── /portfolio (배당 포트폴리오)
├── /goal (목표 및 시뮬레이션)
├── /settings (설정)
└── /auth
    ├── /login
    └── /signup
```

---

## 화면별 상세

### 1. 로그인/회원가입 (/auth/*)
- **로그인**: 이메일+비밀번호, 소셜(Google 등) 로그인
- **회원가입**: 이메일, 비밀번호, 비밀번호 확인

### 2. 메인 대시보드 (/)
- 이번 달 씨앗돈 (절약액) 카드
- 씨앗 N알 수확 시각화 (게이미피케이션)
- 목표 월 배당금 진행률 바
- "목표 달성까지 약 N개월" D-day
- 스트릭/연속 기록 배지
- 최근 지출 5건 미리보기
- FAB: 지출 추가 버튼

### 3. 지출 목록 (/expenses)
- 월별 필터
- 카테고리별 필터
- 지출 목록 (금액, 카테고리, 일자, 메모)
- 수정/삭제

### 4. 지출 추가 (/expenses/new)
- 금액 입력
- 카테고리 선택
- 일자 선택 (기본: 오늘)
- 메모 (선택)

### 5. 예산 설정 (/budget)
- 연-월 선택
- 카테고리별 예산 입력 폼
- 저장

### 6. 배당 포트폴리오 (/portfolio)
- 현재 보유 종목 목록
- 현재 월 배당금 합계
- 종목 추가/수정/삭제

### 7. 목표 및 시뮬레이션 (/goal)
- 목표 월 배당금 입력
- 추가 월 납입액 입력 (씨앗돈 자동 반영)
- 시뮬레이션 결과: 달성 예상 기간

### 8. 설정 (/settings)
- 프로필
- 로그아웃
