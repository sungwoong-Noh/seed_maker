- [v] 예산/목표 페이지: 카드 호버 시 그림자

### 3. 버튼 클릭 피드백

### 4. 접근성
- [v] **Tab** 키로 포커스 이동 시 SkipLink "본문으로 건너뛰기" 먼저 포커스

- [v] Input 라벨과 입력창 **htmlFor/id** 연동 확인
- [v] 에러 시 `aria-invalid` 및 안내 문구 `aria-describedby` 확인

### 5. 예산 vs 실지출 차트

- [v] 대시보드에서 "카테고리별 예산 vs 실지출" 차트 표시 확인
- [v] 막대: 파란색(예산), 회색(실지출)
- [v] Tooltip 호버 시 금액 원화(₩) 포맷 확인
- [v] Legend(예산/실지출) 표시 확인
- [v] 카테고리가 0건이면 차트 섹션 비표시
