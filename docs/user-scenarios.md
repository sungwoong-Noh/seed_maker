# Seed Maker 사용자 시나리오

## 타깃 유저
- 30대 FIRE 추구자, 배당 성장 투자에 관심 있는 직장인
- 기존 가계부 앱 사용에 실패한 경험 (지루함, 기록 중단)
- "아낀 돈을 배당주에 넣으면 언제쯤 월 100만원 배당이 될까?" 같은 질문에 답을 얻고 싶어함

---

## 시나리오 1: 첫 진입 및 온보딩

**상황**: 앱을 처음 설치한 사용자

1. 앱 실행 → 로그인/회원가입 화면
2. 이메일 또는 소셜 로그인으로 간편 가입
3. 온보딩 (선택): "목표 월 배당금이 얼마인가요?" → "이번 달 예산을 세워볼까요?" → 카테고리별 예산 입력 유도
4. 메인 대시보드 진입

**목표**: 최소 단계로 빠르게 시작할 수 있게 함

---

## 시나리오 2: 일상 지출 기록 (핵심)

**상황**: 점심 식사 후 카드 결제 8,500원

1. 앱 메인 또는 플로팅 버튼으로 "지출 추가" 열기
2. 금액 8,500원 입력
3. 카테고리 선택: 식비 (기본값 또는 최근 사용)
4. (선택) 메모: "점심"
5. 저장 → 즉시 씨앗돈 대시보드 반영

**목표**: 3~5초 안에 기록 완료. 입력 필드 최소화

---

## 시나리오 3: 월 예산 설정 및 조정

**상황**: 새 달 시작, 이번 달 예산을 세우려 함

1. 설정 또는 예산 메뉴 진입
2. 카테고리별 예산 입력 (식비 30만, 교통 10만, 공과금 15만 등)
3. 저장 → 해당 월 대시보드에 예산 대비 실지출 시각화

**목표**: 직관적인 예산 입력, 이후 달 복사 기능 (Nice-to-have)

---

## 시나리오 4: 씨앗돈 확인 및 동기 부여

**상황**: 한 달 지출 기록 후, 얼마나 아꼈는지 확인

1. 메인 대시보드에서 "이번 달 씨앗돈" 확인
2. 씨앗 X알 수확 (예: 15만원 절약 = 씨앗 15알) 시각화
3. 카테고리별 예산 vs 실지출 비교
4. "이번 달 vs 지난달" 절약률 비교 (게이미피케이션)

**목표**: 절약의 성과를 즉시 인지, 기록 동기 부여

---

## 시나리오 5: 배당주 포트폴리오 입력

**상황**: VOO 10주, SCHD 5주 보유. 이걸 앱에 반영하려 함

1. 배당/포트폴리오 메뉴 진입
2. "종목 추가" 클릭
3. 종목명, 보유 수량, 주당 배당금(또는 배당 수익률), 현재 주가 입력
4. 저장 → 현재 월 배당금 자동 계산

**목표**: MVP에서는 수동 입력. 이후 API로 자동화

---

## 시나리오 6: 목표 월 배당금 및 시뮬레이션

**상황**: "월 100만원 배당 목표, 언제쯤 도달할까?"

1. 목표 설정 화면에서 "목표 월 배당금: 1,000,000원" 입력
2. 추가 월 납입 가능액: 씨앗돈(자동) + 기타(직접 입력)
3. 시뮬레이션 실행
4. "목표 달성까지 약 N개월" + 진행률 바 확인

**목표**: 게임처럼 "언제 끝나는지" 보여주어 동기 부여

---

## 시나리오 7: 연속 기록(스트릭) 보상

**상황**: 7일 연속 지출 기록 중

1. 대시보드 상단 또는 배지 영역에 "7일 연속 기록!" 메시지
2. 배지 또는 시각적 보상
3. 기록 유지 동기 부여

**목표**: 앱 재방문 및 지속적 기록 유도
